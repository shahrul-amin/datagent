# Datagent

A Google Gemini-inspired data science agent built with React and Tailwind CSS.

## Features

- 💬 Gemini-like chat interface
- 📁 File attachments (images, PDFs, text files)
- ✨ Real-time streaming responses
- ⚡️ Responsive design

## Prerequisites

- npm (v9+)
- Google Gemini API key

## Installation

1. **Clone the repository**
```bash
git clone https://github.com/shahrul-amin/datagent.git
cd gemini-clone-react
```
2. **Install dependencies**
```bash
npm install
```

3. **Set up environment variables**
Create .env file in root directory:
```bash
VITE_GEMINI_API_KEY=your_api_key_here
```

## Running The App
```bash
npm run dev
```
